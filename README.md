
# Application HRNet

Application de gestion des ressources humaines construite avec React, TypeScript et Vite.

## Technologies utilisées

- **React 18** - Bibliothèque d'interface utilisateur
- **TypeScript** - Typage statique pour JavaScript
- **Vite** - Outil de build rapide et serveur de développement
- **Zustand** - Gestion d'état légère
- **React Hook Form** - Gestion des formulaires
- **Zod** - Validation des schémas de données

## Installation

1. **Cloner le repository**
   ```bash
   git clone https://github.com/lanageuse/projet-15-HRNet.git
   cd projet-HRNet
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer l'application en développement**
   ```bash
   npm run dev
   ```

4. **Ouvrir dans le navigateur**
   ```
   http://localhost:5173
   ```

## Fonctionnalités

- **Création d'employés** - Formulaire de saisie avec validation
- **Liste des employés** - Tableau interactif avec tri par colonne
- **Interface responsive** - Adaptée aux différentes tailles d'écran
- **Gestion d'état centralisé** - Utilisation de Zustand
- **Validation des données du formaulaire** - Contrôle des saisies avec Zod et React Hook Form
